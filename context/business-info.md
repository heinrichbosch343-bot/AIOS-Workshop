## Organization Overview

Heinrich runs a solo AI agency that builds custom **AI Operating Systems (AIOS)** for medium to large companies. The service delivers a fully personalized AI intelligence layer for each client — using the AAA Accelerator framework to install, configure, and maintain a modular stack of AI systems tailored to the client's business.

The agency has no specific industry focus at launch. The first build is Heinrich's own workspace, used to refine the process before taking it to paying clients.

## Products / Services / Focus Areas

- **AIOS Build (Setup)** — Full installation and configuration of a client's AI Operating System, including all relevant modules. Priced at $2,500–$5,000 as a one-time setup fee.
- **Monthly Retainer (Maintenance)** — Ongoing maintenance of the client's AIOS stack. Retainer scales upward as new modules are added.
- **Module Additions** — Clients can request new systems to be added to their stack at any time, increasing their retainer accordingly.

### The Standard AIOS Stack per Client

Each client receives a customized version of the following modules:

| Module | Purpose |
|--------|---------|
| Context OS | Builds Claude's understanding of the client's business |
| Infra OS | Version control, GitHub backup, workspace hygiene |
| Data OS | Connects and surfaces the client's live business data |
| Intel OS | Daily briefing, signals monitoring, intelligence synthesis |
| Command OS | Telegram integration — client interacts with their AIOS via Telegram |

## Key Context

- **Stage:** Pre-revenue, just launched. First client is Heinrich himself (test build).
- **Team:** Solo operator — Heinrich runs everything.
- **Sales motion:** Warm referrals to land first clients, then automated cold outreach via Instantly AI (sourcing leads from Apollo).
- **Pricing model:** Setup fee + monthly retainer that grows with the client's stack complexity.
- **Differentiator:** Fully bespoke AIOS per client — not a generic tool, but a custom-built AI layer that understands and runs their specific business.
